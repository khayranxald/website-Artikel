# Changelog

All notable changes to NIMpress will be documented in this file.

## [1.0.0] - 2026-01-26

### Added - Minggu 1 (Setup Project)
- Laravel 12 installation
- Tailwind CSS with dark mode
- Alpine.js integration
- Glassmorphism UI design
- Basic layout structure

### Added - Minggu 2 (Authentication)
- Custom login using NIM
- Register mahasiswa
- Role-based authentication (Admin & Mahasiswa)
- Dark mode toggle
- Glassmorphism login/register pages

### Added - Minggu 3 (Database & Models)
- Posts, Categories, Comments, Likes tables
- Eloquent relationships
- Database seeders & factories
- Category system
- Enhanced animations

### Added - Minggu 4 (CRUD Artikel)
- Create, Read, Update, Delete articles
- Upload thumbnail
- Rich text editor
- Draft & publish status
- Public article pages
- AJAX like system
- Comment system

### Added - Minggu 5 (Moderasi & Admin)
- Admin dashboard with statistics
- Comment moderation (approve/reject/delete)
- Post management (status change, delete)
- User management
- Notification badge for pending comments
- Delete comment by author/admin

### Added - Minggu 6 (Profile & Enhancements)
- AJAX like without reload
- Profile management (edit, avatar upload)
- Change password
- Public profile page
- User statistics
- Author linking system

### Added - Minggu 7 (Dark Mode & Responsive)
- Enhanced dark mode with database sync
- Toast notification system
- Scroll to top button
- Loading states & skeleton screens
- Responsive components
- Mobile optimization
- Accessibility improvements

### Added - Minggu 8 (Final Polish)
- Export artikel to PDF (DomPDF)
- Custom error pages (404, 403, 500)
- SEO meta tags
- Performance optimization
- Database indexing
- Code documentation
- README.md
